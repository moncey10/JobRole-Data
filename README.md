# Data Governance and ML Pipeline

This repository contains the folder structure and resources for Sprint 1 tasks:

- **data/raw**: Raw dataset files.
- **data/processed**: Cleaned and preprocessed data.
- **models/baseline**: Baseline machine learning models.
- **models/advanced**: Advanced machine learning models.
- **scripts/data_preprocessing**: Scripts for data cleaning and transformation.
- **scripts/model_training**: Scripts for training machine learning models.
- **notebooks**: Jupyter notebooks for exploratory analysis.
- **config**: Configuration files for data validation and model hyperparameters.
